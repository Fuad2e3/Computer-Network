**Empowering Connectivity through robust architecture and efficient routing.**

The **Campus Network** project is a comprehensive network infrastructure design developed using **Cisco Packet Tracer**. It simulates a modern university campus environment, focusing on scalability, security, and high-speed data transmission.

---

## 🚀 Key Features

*   **VLAN Implementation:** Segmented network traffic for different departments (Administration, Students, Faculty).
*   **Inter-VLAN Routing:** Seamless communication between subnets using Layer 3 switches/routers.
*   **Dynamic Routing:** Configured with protocols (OSPF/EIGRP) for optimized path selection.
*   **Security Protocols:** Access Control Lists (ACLs) and Port Security to protect the network.
*   **Wireless Connectivity:** Integrated Wi-Fi access points for campus-wide mobility.
*   **Server Services:** Simulated DNS, DHCP, and HTTP services for a full-stack network experience.

## 📁 Project Structure

The project files are organized to show the evolution of the network design:

```text
.
├── network-designs/
│   ├── final-campus-network.pkt    # The complete, fully-featured network design.
│   ├── v1-basic-topology.pkt       # Initial structural layout and basic connectivity.
│   └── v2-advanced-routing.pkt     # Intermediate version with routing and VLANs.
├── LICENSE                          # MIT License details.
└── README.md                       # This documentation.
```

## 🛠️ Getting Started

To explore or modify the network designs:

1.  **Download Cisco Packet Tracer:** Ensure you have the latest version installed.
2.  **Clone this Repository:**
    ```bash
    git clone https://github.com/fuadk/Computer-Network.git
    ```
3.  **Open Files:** Navigate to the `network-designs/` folder and open the `.pkt` files directly in Packet Tracer.
